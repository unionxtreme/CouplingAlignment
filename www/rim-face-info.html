Rim Face Info

Pre Check

  1. Machine LOTO
  2. Straight edge align
  3. Check bracket sag
  4. Correct soft foot
  5. Mark clock positions
  6. Check machine rotation
  7. Check record run out

Alignment

  1. Indicator half travel
  2. Record clock positions thru driver
  3. Rotate couplings together
  4. Record readings
  5. Make alignment adjustments
  6. Repeat until in tolerance
